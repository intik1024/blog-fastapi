# blog-fastapi

## Описание

Это блог в котором можно содовать,удалять и комментировать посты

## Возможности

- Создание постов
- Удаление постов
- Коментирование постов
- Добавление тегов для постов

## Установка:
# Клонируйте репозиторий
git clone https://github.com/intik1024/blog-fastapi.git
# Перейдите в папку проекта
cd blog-fastapi
# Установите зависимости
pip install -r requirements.txt
